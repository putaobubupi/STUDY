第二章 · 波形变"歪"了

同样的正弦电压，不同的负载画出完全不同的电流

✓ 线性负载

电流 ∝ 电压 · 波形完美正弦

白炽灯 · 电阻炉 · 电热器

✗ 非线性负载

电流仅在电压峰值附近脉冲导通

开关电源 · 变频器 · 充电器

💡 核心洞察

非线性负载内部有整流桥 + 电容。只有当交流电压瞬时值 > 电容电压时，二极管才导通，出现短促的大电流脉冲。大部分时间电流 = 0。

傅里叶变换告诉我们：脉冲电流 = 基波 + 3次谐波 + 5次谐波 + 7次谐波 + ...

第三章 · 谁制造了谐波

三大谐波源逐一拆解——每种设备产生不同"味道"的谐波

⬡

电力电子开关型

占比 ~40%

利用晶闸管/IGBT 等开关器件进行整流或逆变，电流被"切"成缺角波形。

典型设备：变频器、整流器、UPS、开关电源、充电桩

特征谐波： h = 6k ± 1（5次、7次、11次、13次...）

⧉

铁磁饱和型

占比 ~15%

变压器铁芯的磁化曲线是非线性的。电压偏高时铁芯进入饱和区，励磁电流变成尖顶波。

典型设备：电力变压器、电抗器（轻载时更严重）

特征谐波： 3次、5次为主，3次最突出

⚡

电弧型

占比 ~20%

电弧的伏安特性高度非线性——起弧后电流很大但电压几乎不变，波形不规则且随机性强。

典型设备：电弧炉、电焊机、气体放电灯

特征谐波： 2~7次 + 间谐波（非整数倍）

第五章 · 谐波的影响

谐波电流进入设备后会发生什么？

🔥 变压器过热与噪声

谐波频率高 → 集肤效应增强 → 绕组电阻增大 → I²R 损耗增加；涡流损耗 ∝ f²

🔄 电动机的反向刹车

5次谐波（负序）产生反向旋转磁场 → 制动转矩；7次谐波产生超高速正向磁场 → 振动

⚡ 中性线过载

3次谐波三相中同相位，中线电流 = Ia₃ + Ib₃ + Ic₃ = 3倍

📈 电容谐振放大

电容对高频谐波阻抗低（Z=1/2πfC），大量谐波流入，可能形成谐振
